# ☕ Java Full Stack Training

This repository contains materials, notes, exercises, and resources from the **Java Full Stack Development training program**.

The goal of this repository is to track learning progress and organize all resources used during the course.

---

## 📚 Course Overview

The training focuses on building full stack applications using **Java technologies** for backend development and modern tools for frontend development.

Main topics covered:

- Java programming fundamentals
- Object-Oriented Programming (OOP)
- Backend development
- Databases and SQL
- Web development
- Full Stack application architecture
- Version control with Git
